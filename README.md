# LBPH Face Recognition with OpenCV

Face recognition system using Local Binary Pattern Histogram (LBPH) and OpenCV.

## Features

* Face detection
* Face recognition
* Real-time testing
* Performance evaluation

## Technologies

* Python
* OpenCV
* NumPy

## Project Purpose

This project was developed as part of a research study on the effect of lighting conditions on the accuracy of LBPH face recognition.

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python capture.py
```
